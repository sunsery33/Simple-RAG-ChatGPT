# 智能客服知识库示例

## 产品简介

Simple RAG ChatGPT 是一个基于 API 和 RAG 的简易智能问答系统。它可以上传企业资料、课程资料或项目文档，把文本切成片段后生成向量索引，并在用户提问时检索最相关的资料作为上下文。

## 支持的文件

当前版本支持 `.txt`、`.md`、`.pdf` 和 `.docx` 文档。上传后系统会抽取文本、清洗内容、按长度切块，并调用 Embedding API 生成向量。

## 回答策略

当知识库中存在相关资料时，系统会优先根据资料回答，并在答案中用 `[1]`、`[2]` 这样的格式引用来源。如果资料不足，系统需要明确说明知识库中没有直接依据，再给出一般建议。

## 适用场景

- 企业内部制度问答
- 课程资料问答
- 产品说明书问答
- 实习项目或毕业设计演示
- 个人知识库助手

## 后续规划

后续可以把本地 JSON 向量库替换为 pgvector、Qdrant、Milvus 或 Chroma，也可以加入用户登录、多知识库管理、流式输出和引用高亮。
